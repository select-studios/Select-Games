Whats Vandal?

VANDAL is a game based off the MFPS package built by LOVATTO. In 2020 Polyshot's development began. this was shortly ended in july 2021 due to another game studio (TEKNOSOFT) Announced a game to steam, also called Polyshot. not only did it have the same name but the game was basically the same. we wanted to be as unique as possible this meant canceling Polyshot and taking it off steam
in 2022 Founder Mellurboo requested we restart the development under a new name and a game plan so VANDAL began, the progress has been so rewarding and we love how its going, with a public beta releasing soon along with the select launcher, we couldn't be more happy.
